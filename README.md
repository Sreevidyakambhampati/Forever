# Forever

### How to run ?

1. Create .env files for backend, admin and frontend.
2. run `npm install` for backend, admin and frontend.
3. First run the backend server.
   * change working directory to backend using  `cd backend`
   * run the server using `npm run server`
4. Run the frontend server.
   * Change the working directory to frontend using  `cd frontend`
   * run the server using `npm run dev`
5. Run the admin server.
   * Change the working directory to admin using `cd admin`
   * run the server using `npm run dev`
